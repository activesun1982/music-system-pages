Privacy Policy

Last updated: April 13, 2026

1. Overview

Music System Worker D is a personal content distribution tool operated by Andrew Liew. This privacy policy explains how the application handles data in connection with social media platform integrations.

2. Data Collection

This application does not collect, store, or process personal data from any third-party users. The application is used solely by its owner to distribute original music content. The only data handled is: OAuth authentication tokens required for platform API access (stored locally, never transmitted to third parties), original music content metadata (titles, descriptions, tags) created by the operator, and platform response data (post IDs, URLs) for record-keeping purposes.

3. Data Storage

All data is stored locally on the operator's personal computer. No cloud databases, external servers, or third-party storage services are used. Authentication credentials are stored in local JSON files and are never committed to version control or shared.

4. Third-Party Services

This application integrates with TikTok, YouTube, and Facebook APIs solely for the purpose of publishing original music content. The application accesses only the minimum required API scopes for content posting. No user data is read from these platforms beyond what is necessary for the posting workflow.

5. Data Sharing

No data is shared with third parties. The application does not use analytics services, advertising networks, or data brokers.

6. Data Retention

Authentication tokens are retained locally until manually revoked or expired. Content posting records are retained locally for the operator's personal record-keeping.
